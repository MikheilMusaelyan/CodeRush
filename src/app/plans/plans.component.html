Front End Development (20 weeks)

Weeks 1-2 : Introduction to HTML and CSS
  • Learn the basics of HTML tags, attributes, and structure
  • Understand CSS selectors, properties, and styling techniques
    • Weeks 3-4 : Intermediate HTML and CSS
  • Dive deeper into HTML5 features and semantic markup
  • Explore advanced CSS concepts such as flexbox and grid layouts
    • Weeks 5-6 : Responsive Web Design
  • Understand the principles of responsive web design
  • Learn how to create mobile-friendly layouts using media queries and CSS frameworks
    • Weeks 7-8 : JavaScript Fundamentals
  • Cover the fundamentals of JavaScript programming language
  • Learn about variables, data types, functions, and control flow
    • Weeks 9-10 : Advanced JavaScript & Project
  • Explore more complex JavaScript concepts like objects, arrays, and prototypes
  • Learn about DOM manipulation and event handling
  • Create a real life project with HTML, CSS and JavaScript
    • Weeks 11-13 : Introduction to Front End Frameworks
  • Dive into popular front-end framework like Angular
  • Learn the basics of component-based development
    • Weeks 14-18 : Advanced Front End Framework Concepts
  • Explore the fundamentals of State Management
  • Learn the importance of testing in web development
  • Cover the basics of Server Side Rendering
  • Understand component lifecycle and data flow
    • Weeks 19-20: Final Project
  • Create a Final Project and enhance your portfolio
  • Engage in open discussions and seek clarification on concepts
